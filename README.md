# INVERSE-OF-A-MATRIX
## Aim:
To write a python program to find the inverse of a matrix
## Equipment’s required:
1. 	Hardware – PCs
2. 	Anaconda – Python 3.7 Installation / Moodle-Code Runner
## Algorithm:
### Step1 :
 Start the program.
### Step 2:
 Write the codings.
### Step 3:
 Print the programs.
### Step 4: 
End the program.

## Program:
~~~
import numpy as np
A=np.array([[2,1,1],[1,1,1],[1,-1,2]])
ans=np.linalg.inv(A)
print(ans)
~~~
